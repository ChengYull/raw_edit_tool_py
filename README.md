# RawEditTool

raw图编辑工具，支持raw图的直接编辑操作，可直接在raw图上进行手动绘制

## 更新

- 2026-02-10：

初版上传

完成了图像绘制paintWidget组件，可以显示任意格式的图片，并且支持在图片上进行绘制操作，支持在图像上进行绘制

增加了拖拽支持，可以直接将图片拖入界面中进行编辑

增加了通过鼠标移动事件，以及鼠标点击事件进行绘制操作，支持在图像上进行绘制

增加了保存功能，原图为raw则保存为raw格式，原图为其他格式则保存为png格式

- 2026-02-25:

增加了RGB与Gray模式手动切换

增加笔刷大小设置，暂定范围3-30

增加了重置按钮
